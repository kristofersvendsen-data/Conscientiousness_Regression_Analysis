
This exploratory analysis using the bfi data set from the "psych" package examines which demographic-based predictors are associated with higher scores on a conscientiousness measure.

A detailed description of the analysis is provided in an R Markdown (.Rmd) file. The original file is provided in this repository and can be opened with RStudio; however, the output of this file can be viewed [here](https://kristofersvendsen-data.github.io/Conscientiousness_Regression_Analysis/). Additionally, a slide deck highlighting background information and key results is provided in the repository.
