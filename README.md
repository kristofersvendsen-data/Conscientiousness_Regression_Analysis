
This exploratory analysis using the bfi data set from the "psych" package examines which demographic-based predictors are associated with higher scores on a conscientiousness measure.

A detailed description of the analysis is provided in an R Markdown (.Rmd) file. The original file is provided in this repository and can be opened with RStudio; however, the output of this file can be viewed [here](https://kristofersvendsen-data.github.io/Conscientiousness_Regression_Analysis/). Additionally, a [slide deck](https://github.com/kristofersvendsen-data/Conscientiousness_Regression_Analysis/blob/main/Presentation_Factors_Affecting_Conscientiousness.pdf) highlighting background information and key results is provided in the repository.

This analysis involved data cleaning and preparation followed by score calculation. Reliability measures are discussesd and both Chronbach's alpha and McDonald's hierarchical omega are calculated with boostrapped 95% confidence intervals. Multiple regression was performed and the results contextualized and presented. Visualizations of the data are also prepared.
